# 🚗 Desafio Ford Enter - Landing Page Ford Ranger

## 📖 Sobre o Projeto
Este projeto foi desenvolvido como parte do desafio prático de HTML e CSS do programa educacional **Ford Enter**. O objetivo principal do desafio foi aplicar conceitos fundamentais de front-end na construção de uma landing page completa, focando em marcação semântica, estruturação de layout com Flexbox e a criação de um design responsivo.

A aplicação simula uma interface de vitrine para as gerações e versões da Ford Ranger, garantindo uma navegação fluida e uma excelente experiência de usuário (UX) tanto em computadores quanto em dispositivos móveis.

## ✨ Funcionalidades
* **Navegação Multi-páginas:** Estrutura interligada contendo Página Inicial, Lançamentos e Contato, com rotas relativas corretamente configuradas.
* **Vitrine e Comparativo (Cards):** Seção de seleção de modelos da picape (XL, XLS e Storm) utilizando um sistema de grid/flexbox padronizado para alinhar perfeitamente imagens, textos e caixas de seleção, independentemente da quantidade de conteúdo.
* **Formulário de Contato Acessível:** Campos de entrada estilizados com feedback visual imediato ao usuário através de pseudoclasses CSS (`:focus`, `:valid`, `:hover`).
* **Design Responsivo:** Layout que se adapta automaticamente a telas menores (smartphones e tablets) utilizando Media Queries e controle dinâmico de espaçamentos.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação de marcação semântica (`<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<footer>`).
* **CSS3:** 
  * Layout responsivo utilizando **Flexbox** (`display: flex`, `align-items`, `justify-content`).
  * Controle avançado de espaçamentos, crescimento e compressão de elementos (`gap`, `flex-shrink`, `flex-grow`, `margin: auto`).
  * Ajuste e padronização de imagens com `object-fit`.
  * Efeitos de transição (`transition`) para interatividade visual.

## 🚀 Como Executar o Projeto

Como o projeto foi construído utilizando caminhos relativos, ele pode ser executado facilmente de duas maneiras:

**Opção 1: Visualização direta (Local)**
1. Clone este repositório ou faça o download da pasta do projeto (`.zip`).
2. Extraia os arquivos e dê um duplo clique no arquivo `index.html`.
3. O projeto abrirá e funcionará perfeitamente no seu navegador padrão.

**Opção 2: Utilizando Servidor Local (Recomendado para edição)**
1. Abra a pasta do projeto no **Visual Studio Code**.
2. Instale a extensão **Live Server**.
3. Clique com o botão direito no arquivo `index.html` e selecione **"Open with Live Server"**.
